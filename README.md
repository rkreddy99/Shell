# Shell from scratch in C

## Commands supported
* ls (no flags) dir(s)
* grep (no flags) pattern, files(s)
* cat (no flags) file(s)
* mv (no flags) file to dir / files to dir / dir to dir
* cp (no flags) file to file / files to dir/ dir to dir
* pwd (no flags) dir
* rm file / [-r] dir
* chmod (no flags)
* mkdir (no flags) dir(s)
